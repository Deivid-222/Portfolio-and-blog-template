-- Requirements --

1. Install Node.js on your system.
2. Run this command on your terminal "npm install".
3. Every time you open it for the first time, run this command so that the tailwind code compiles and becomes css "node --run build:styles".
4. Write this command on your terminal to be able to use that animations that i have used "npm install @midudev/tailwind-animations".
